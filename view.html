The Color Mixer

Background

The usage of HTML, JavaScript and CSS are undoubtedly ubiquitously across the modern internet. One of the major attributes used across all three formats is the hexadecimal string representation of color.

More info on Web Colors

Exercise

Using fetch(), retrieve two starting colors from the remote server by substituting your email into the provided link here. After fetching the two color values, the decoder function will produce a secret color by first averaging the colors and then inverting that result. Finally, your code should color in the provided circles with the appropriate colors and list their hex codes.

The secret color is produced in two steps:

  1. Average the two starting colors rounding down.
    • Each color channel byte should be calculated independently of one another.
    • For these hex inputs, you should end up with this Mixed result:
      • #123456  #ABCDEF  Mixed
R 12     R AB     R 5E
G 34     G CD     G 80
B 56     B EF     B A2
  2. Invert the resulting color from step one.
    • Each color channel byte or hex character should be calculated independently.
    • For this hex input, you should end up with this Secret result:
      • #5E80A2  Secret
R 5E     R A1
G 80     G 7F
B A2     B 5D

You may use anything built into the JavaScript programming language, such as base conversion, but you can also do it manually. Keep all changes in the JavaScript + jQuery section only.

Remote API

URL: Call this endpoint by supplying your email at the end, replace :email with your email address; it is a path parameter. 
https://eworldes-color-mixer.cyclic.app/colors/:email
Model: 
{
    "color1": "808000",
    "color2": "A0522D"
}
